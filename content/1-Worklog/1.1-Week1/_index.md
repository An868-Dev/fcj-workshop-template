### Week 1 Achievements:

* Successfully connected with FCJ members and clearly understood internship expectations, workflows, and working procedures.

* Learned the basic concepts of core AWS service groups, including:

  * Compute: EC2 and an introduction to Lambda

  * Storage: S3 and EBS

  * Networking: VPC fundamentals, subnets, and Internet Gateway

  * Database: basic overview of Amazon RDS

  * Security: IAM and account-level security

* Created and set up an AWS Free Tier account for hands-on practice.

* Became familiar with the AWS Management Console, including how to navigate, search for services, and manage resources.

* Installed and configured the AWS CLI, including setting up:

  * Access Key

  * Secret Key

  * Default region

  * Output format

* Performed basic AWS CLI tasks such as:

  * Verifying IAM identity and CLI configuration

  * Listing available AWS regions

  * Viewing EC2 resources

  * Creating and managing key pairs

  * Checking active services and running resources

  * Completed several introductory AWS hands-on labs, gaining practical experience with both the AWS Console and CLI.

  * Built a strong foundation to work with more advanced AWS services in the following weeks.
